Welcome to Simple Image Microservice

This app searches for images on Unsplash

To perform a new search:

  • Please append a search term to the address of this page, followed by "/search/"
  • If you wish to view further results add a page offset to the end of the search query, followed by ?page=

To view the latest searches:

Append /latest to the page address

Example use:

To search for images of iguanas use the following

https://simpleimgfinder.herokuapp.com/search/iguanas

To search for images of iguanas but skip to the second page of results:

https://simpleimgfinder.herokuapp.com/search/iguanas?page=2

Finally, to view the last ten queries to this app type

https://simpleimgfinder.herokuapp.com/latest
